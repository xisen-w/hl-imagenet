# Phase 1 Code Snapshot

This folder freezes the Phase 1 symbolic classifier before the live codebase is adapted for Phase 2.

Phase 1 used:
- 4 real Tiny ImageNet classes: `golden_retriever`, `mushroom`, `teapot`, `school_bus`
- 6 synthetic placeholder classes: `banana`, `bicycle`, `eagle`, `laptop`, `piano`, `zebra`
- A 230-image development set used throughout iterative tuning

The live package may change for Phase 2. Treat this folder as a read-only reference for the Phase 1 classifier, feature library, sensors, evaluator, and supporting types.
